## 🔧 담당 구현 영역
- **Header**
  - 글로벌 네비게이션 구조 설계
  - 메뉴 영역 레이아웃 및 인터랙션 처리

- **Footer**
  - 정보 영역 구조화 및 공통 레이아웃 구현

- **Series 섹션**
  - 시리즈 콘텐츠 슬라이드 UI 구현
  - **외부 라이브러리 없이 JavaScript만으로 슬라이드 로직 직접 구현**
  - 버튼 제어 및 콘텐츠 전환 흐름 구현

## 📌 구현 포인트
- HTML 구조를 먼저 설계한 뒤 CSS로 레이아웃을 정리하고,  
  JavaScript를 활용해 **화면 흐름과 사용자 인터랙션을 단계적으로 구현**
- 추후 반응형 대응 및 코드 리팩토링을 고려해 **확장 가능한 구조**로 작업 중
